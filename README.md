# Matlab C/C++ Coder Example

This sample explains how to import C/C++ code generated by Matlab Coder in CMake Project

### Prerequisites

* Create Matlab Function
* Install Matlab Coder Add-on

### How it works

* Generate C/C++ code
* Generate .zip pkg
* Import .zip package in existing CMake Project
* Modify CMakeLists.txt so to include directories and link executable with source files and headers.

### Installing

* Clone repository
* Load CMake Project
* Build and Run HelloWorld
